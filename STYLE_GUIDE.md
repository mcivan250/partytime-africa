# PartyTime Africa UI/UX Style Guide: Luxury Minimalist (Kardashian Aesthetic)

This style guide outlines the design principles, color palette, typography, and UI components to achieve a high-end, luxury minimalist aesthetic for the PartyTime Africa application, inspired by brands like SKIMS and SKKN by Kim.

## 1. Core Design Principles

*   **Minimalism**: Clean, uncluttered interfaces with ample whitespace to emphasize content and create a sense of sophistication.
*   **Elegance**: Subtle animations, refined transitions, and polished elements that convey a premium feel.
*   **Focus on Content**: Prioritize event visuals and information, making them the heroes of the design.
*   **Intuitive Navigation**: Streamlined user flows and clear calls to action.
*   **Inclusivity**: Design choices that appeal to a diverse audience while maintaining a luxurious tone.

## 2. Color Palette

The primary color palette will consist of warm, earthy neutrals, complemented by deep, rich accents. This creates a sophisticated and inviting atmosphere.

| Color Name       | HEX Code   | Usage                                                              |
| :--------------- | :--------- | :----------------------------------------------------------------- |
| Primary Background | `#1A1A1A`  | Dark mode background, primary surfaces.                            |
| Secondary Background | `#2C2C2C`  | Card backgrounds, secondary sections.                              |
| Text Primary     | `#FFFFFF`  | Main text, headings.                                               |
| Text Secondary   | `#B0B0B0`  | Subheadings, descriptions, inactive states.                        |
| Accent Gold      | `#C5A35A`  | Call-to-action buttons, highlights, important icons.               |
| Accent Nude      | `#E0DCD7`  | Subtle highlights, selected states, secondary buttons.             |
| Error Red        | `#FF4D4D`  | Error messages, destructive actions.                               |
| Success Green    | `#4CAF50`  | Success messages, positive feedback.                               |

## 3. Typography

We will use a combination of a modern sans-serif for headings and a clean, readable sans-serif for body text. This pairing ensures clarity and a contemporary feel.

*   **Headings (H1, H2, H3)**: **Montserrat** (or similar geometric sans-serif)
    *   Weight: Semi-Bold to Bold
    *   Usage: Event titles, section headers, prominent text.

*   **Body Text (Paragraphs, Labels)**: **Inter** (or similar humanist sans-serif)
    *   Weight: Regular to Medium
    *   Usage: Event descriptions, form labels, general information.

*   **Call-to-Action/Buttons**: **Inter** (or similar humanist sans-serif)
    *   Weight: Medium to Semi-Bold
    *   Usage: Buttons, navigation links.

## 4. UI Components & Elements

### Buttons

*   **Primary Button**: Solid background with `Accent Gold`, white text. Rounded corners (small radius).
*   **Secondary Button**: Transparent background, `Accent Gold` border, `Accent Gold` text. Rounded corners.
*   **Ghost Button**: Transparent background, `Text Secondary` border, `Text Secondary` text. Rounded corners.

### Input Fields

*   **Style**: Minimalist, with a subtle border (`Text Secondary` or `Secondary Background`).
*   **Focus State**: Border changes to `Accent Gold`.
*   **Error State**: Border changes to `Error Red`.
*   **Placeholder Text**: `Text Secondary`.

### Cards

*   **Background**: `Secondary Background`.
*   **Shadow**: Subtle, soft shadow for depth (dark mode).
*   **Content**: Well-spaced, clear typography, prominent images.

### Navigation

*   **Top Navigation**: Minimalist, with logo and essential icons (notifications, messages, profile).
*   **Bottom Navigation (Mobile)**: Clean, icon-based, with active state indicated by `Accent Gold`.

### Imagery

*   **Event Posters**: High-quality, full-bleed images. Focus on striking visuals.
*   **Avatars/Profile Pictures**: Circular, clean, and professional.
*   **Icons**: Simple, line-based icons that align with the minimalist aesthetic.

## 5. Layout & Spacing

*   **Whitespace**: Generous use of whitespace to create a sense of luxury and reduce visual clutter.
*   **Grids**: Consistent grid system for alignment and organization of content.
*   **Padding/Margins**: Standardized padding and margin values across components for consistency.

## 6. Animations & Transitions

*   **Subtle & Smooth**: Animations should be fluid, fast, and non-intrusive.
*   **Examples**: Hover effects on buttons, smooth transitions between pages, subtle loading indicators.

This style guide will serve as the foundation for the complete UI/UX redesign of PartyTime Africa, ensuring a cohesive and luxurious user experience.
